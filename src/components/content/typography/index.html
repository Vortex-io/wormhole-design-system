Typography

Documentation and examples for typography, including global settings, headings, body text, lists, and more.

Global settings

Wormhole sets basic global display, typography, and link styles.

There is a config/_reset.scss file that resets the browsers defaults. It also removes padding and margins for every element in the design system to start from a clean base.

The styles of the following references can be found in components/_text-styles.scss, and the global variables are defined in config/_variables.scss.

A wormhole is a speculative structure linking disparate points in spacetime, and is based on a special solution of the Einstein field equations. It can be visualized as a tunnel with two ends at separate points in spacetime (i.e., different locations, different points in time, or both).

Wormholes are consistent with the general theory of relativity, but whether wormholes actually exist remains to be seen. Many scientists postulate that wormholes are merely projections of a fourth spatial dimension, analogous to how a two-dimensional (2D) being could experience only part of a three-dimensional (3D) object.

Usage
There are 8 link styles implemented for links. These styles can be used on the <a> HTML tag, or in any other HTML tag for inline text like <span>, <strong>, <em>, etc (even <div> HTML tags) by adding the .link class to it.
  • Regular link
    HTML tag <a>
    – or –
    class .link
  • Back link
    HTML tag <a> with class .back
    – or –
    class .link and .back together
  • Positive action link
    HTML tag <a> with class .green
    – or –
    class .link and .green together
  • Destructive action link
    HTML tag <a> with class .red
    – or –
    class .link and .red together
  • Non-primary action link
    HTML tag <a> with class .black
    – or –
    class .link and .black together
  • Small link
    HTML tag <a> with class .small
    – or –
    class .link and .small together
  • Big link
    HTML tag <a> with class .big
    – or –
    class .link and .big together
  • Disabled link
    HTML tag <a> with class .disabled
    – or –
    class .link and .disabled together

    Using JavaScript, you will need to add event.preventDefault(); to the Event Handler to actually disable the link.

Fonts

Both font-family and font-weight references in the design system are handled via SASS variables.

Body font

Halyard Display font is used for body text, normal titles, forms fields, buttons, etc. Almost every element in the design system use this font.

Usage
SASS variable $font-body
class .font-body
Weight variants
There are 3 weight variants implemented:
  • Regular weight (400)
    SASS variable $fw-regular
    class .regular
  • Semi-bold weight (500)
    SASS variable $fw-semi
    class .semi
  • Bold weight (700)
    SASS variable $fw-bold
    class .bold
